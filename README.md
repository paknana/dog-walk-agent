**최종 추천 코스**
**하남 스타필드 실내 코스**

**장소 유형**
실내 복합물

**총 거리**
3.0 km

**예상 산책 시간**
45 분

**안전 점수**
85 점 (매우 안전)

**추천 이유**
- **비·미세먼지 차단**: 실내 환경이므로 비와 미세먼지로부터 강아지를 보호합니다.
- **경사도 낮음**: 경사도 1(가장 완만)로 관절에 부담이 없습니다.
- **포장 상태 우수**: 노면 상태 5(최상)로 미끄러짐 위험이 적습니다.
- **조명·안전**: 조명 수준 5, 안전 점수 85로 야간에도 충분히 밝고 안전합니다.
- **운전 가능**: drive 이동 방식에 최적화된 코스입니다.

**주의사항**
- 실내가 다소 혼잡할 수 있으니, 사람과 다른 반려동물과의 거리를 유지하며 주의하세요.
- 비가 많이 오는 날이므로, 강아지와 함께 사용할 물건(예: 물통, 간식)은 방수 포장해 두면 좋습니다.

이 코스는 비와 미세먼지로 인한 야외 위험을 최소화하면서도 웰시코기의 에너지를 충분히 소모시킬 수 있는 최적의 선택입니다. 즐거운 산책 되세요!
